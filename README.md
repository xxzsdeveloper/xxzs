### 使用方式
1. 选择需要辅助学习的项目，点击开始按钮。
2. 如果应用未登陆，自动跳转到二维码登陆页面，需要使用学习App人工扫码登陆。
3. 登陆后手动切换到学习助手应用，即可辅助学习。
